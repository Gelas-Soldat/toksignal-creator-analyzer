# Contributing

Thanks for improving TokSignal.

## Local setup

No package install is required for the current static version.

```bash
python3 -m http.server 8888
```

Then open `http://localhost:8888`.

## Pull request checklist

| Check | Done |
| --- | --- |
| The app works on mobile width |  |

## Code style

Keep the current MVP simple. Avoid adding a build system unless the feature truly needs it.
